A simple plugin to boost your sales with WooCommerce Upsell and Cross-Sell offers upon purchase of particular products on any page.

== Description ==

Cross/Upsell Popup for WooCommerce is a cross-selling plugin that facilitates store owners to make offer an exclusive product based on customer added cart items through a prompt on any page, enabling them to increase the sales. The offer pops up on customer window after added to cart of particular products set on backend.

== Installation ==

= Automatic installation =

1. Visit **Plugins > Add New**.
2. Search for **Cross/Upsell Popup for WooCommerce**.
3. Install and activate the Cross/Upsell Popup for WooCommerce plugin.

= Manual installation =

1. Upload the entire `cross-up-sell-popup-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Cross/Upsell Popup for WooCommerce plugin.

= After activation =

1. In the Dashboard find **WC Cross Up Sell Popup** menu.
2. Do the settings by following instructions to enable your offer
